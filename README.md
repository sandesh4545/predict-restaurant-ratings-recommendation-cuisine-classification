# Predict-Restaurant-Ratings-Recommendation-Cuisine-Classification
This project focuses on building intelligent data-driven solutions using a real-world restaurant dataset. It combines machine learning, data preprocessing, and recommendation system techniques to extract insights and enhance decision-making in the food and hospitality domain.
# 🍽 Restaurant Analytics & Recommendation System

## 📌 Project Overview
This project focuses on analyzing restaurant data and building machine learning models for:
- Rating Prediction (Regression)
- Restaurant Recommendation System
- Cuisine Classification (Classification)

---

## 🎯 Objectives

### Task 1: Restaurant Rating Prediction
- Predict restaurant ratings using ML models
- Use regression techniques (Linear Regression, Decision Tree)
- Evaluate using MSE and R² score

### Task 2: Restaurant Recommendation System
- Recommend restaurants based on user preferences
- Use content-based filtering
- Criteria: cuisine, price range, etc.

### Task 3: Cuisine Classification
- Classify restaurants into cuisine categories
- Use classification models (Logistic Regression, Random Forest)
- Evaluate using accuracy, precision, recall
